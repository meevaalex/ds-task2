# data-science-intern-task2
Titanic dataset EDA with data cleaning and visualization using python

📌 Overview

This repository contains my submission for Task 2 of the Data Science Internship at Prodigy Infotech.
The objective of this task is to perform Data Cleaning and Exploratory Data Analysis (EDA) on a dataset and identify meaningful patterns, relationships, and trends.

---

📂 Dataset

The dataset used for this project is the Titanic Dataset, which contains information about passengers such as age, gender, class, fare, and survival status.

---

🛠️ Tools and Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

🔍 Data Cleaning

The following preprocessing steps were performed:

- Handled missing values in columns like Age and Cabin
- Removed duplicate records (if any)
- Converted data into appropriate formats for analysis

---

📊 Exploratory Data Analysis (EDA)

Key Analysis Performed:

- Distribution of Age using histogram
- Count of passengers based on Gender
- Survival analysis based on:
  - Gender
  - Passenger Class (Pclass)
- Relationship between Fare and Survival

---

📈 Key Insights

- Female passengers had a higher survival rate compared to males
- Passengers in higher classes (Pclass 1) had better survival chances
- Younger passengers showed different survival patterns

---

📌 Conclusion

This analysis helped in understanding the factors affecting passenger survival on the Titanic.
It demonstrates how data cleaning and visualization can reveal meaningful insights from raw data.
